# HireSense-AI (Smart Recruitment Analytics System)

## Overview

HireSense-AI is an AI-assisted recruitment analytics platform developed as a BCA Major Project. The application helps recruiters analyze resumes, compare candidate profiles with job descriptions, calculate ATS scores, and generate AI-powered hiring insights for better recruitment decisions.

## Features

- Resume Parsing from PDF
- ATS Score Calculation
- Job Description Matching
- Skill Gap Analysis
- AI Candidate Summary
- AI Hiring Recommendation
- AI Interview Questions
- Hiring Workspace Dashboard
- AI Insights Dashboard
- Bulk Resume Processing
- Blind Hiring Support
- SQLite Database Integration

## Technology Stack

- Python
- Streamlit
- SQLite
- Google Gemini AI
- Plotly
- pdfplumber
- ReportLab
